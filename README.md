# Button Collection

### Installation

1. Clone The Project

   ```bash
   git clone https://github.com/Azmye/badrinteractive-test.git
   ```

2. Install dependencies

   ```bash
   npm install or yarn
   ```

3. Start the app

   ```bash
    yarn dev or npm run dev
   ```

   ### Library In Used

   - Tailwind CSS
   - React-Tooltip
